Requisito

Codice: REQ-D-008
Titolo: Formati di export supportati
Categoria: DATI
Priorita: high
Le esportazioni dei log sono funzionalità disponibili a un utente rispetto alle autorizzazioni associate al suo specifico ruolo. Esse devono poter essere prodotte nei principali formati disponibili per le analisi (sicuramente csv/excel) ed eventuali altri formati come JSON e XML, assicurando compatibilità con i sistemi di audit già in uso. I file prodotti devono rispettare gli standard di formato (JSON valido, XML ben formato) in modo che possano essere processati dai sistemi di audit esterni senza necessità di conversioni aggiuntive non previste.

Acceptance Criteria

AC-1L'export dei log è disponibile solo agli utenti autorizzati.
AC-2I file JSON, XML, CSV/excel sono sono validi in base ai relativi standard e contengono le informazioni richieste (evento, timestamp, utente quando presente).
AC-3Un file di esempio per ciascun formato può essere scaricato/esportato dal sistema.
