Requisito

Codice: REQ-I-001
Titolo: Consumo di raccomandazioni prioritarie da sistema esterno con informazioni addizionali
Categoria: INTEGRAZIONE
Priorita: high
Il Motore deve poter ricevere come input raccomandazioni già prioritizzate da un sistema esterno di prioritizzazione basato sulle propensioni d’acquisto. L'utente ha specificato che questi input includono, oltre ai prodotti suggeriti in ordine prioritario, informazioni aggiuntive come il cluster di appartenenza del cliente e le motivazioni della raccomandazione. Questo requisito stabilisce che il motore tratti tali raccomandazioni come input primario esterno e sia in grado di elaborarle secondo le regole interne configurate.

Acceptance Criteria

AC-1È disponibile un canale di integrazione per ricevere una lista ordinata di raccomandazioni da un sistema esterno.
AC-2Il payload di input supporta campi per identificatore prodotto, ordine/priorità, cluster cliente e motivazione della raccomandazione.
AC-3Il motore processa gli input ricevuti e lo fa rispettando le configurazioni locali (es. filtri, matrix, regole di frequenza) definite in altri requisiti.
