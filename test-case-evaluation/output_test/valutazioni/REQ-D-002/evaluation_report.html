Requisito

Codice: REQ-D-002
Titolo: Gestione varianti di descrizione assegnabili ai cluster di clientela
Categoria: DATI
Priorita: medium
Il sistema deve supportare la gestione e l'assegnazione di varianti testuali delle descrizioni delle offerte in funzione di segmenti o cluster di clientela. Il team marketing deve poter associare specifiche varianti a determinati cluster in modo che, quando il cliente appartiene a quel cluster, venga mostrata la relativa variante della descrizione.

Acceptance Criteria

AC-1È possibile creare e salvare più varianti testuali per la stessa offerta nella dashboard marketing.
AC-2Ogni variante può essere associata a uno o più identificatori di cluster/segmento.
AC-3Nei casi di test dove il cliente è contrassegnato con un determinato cluster, l'interfaccia mostra la variante di descrizione corrispondente.
