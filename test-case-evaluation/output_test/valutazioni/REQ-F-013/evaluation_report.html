Requisito

Codice: REQ-F-013
Titolo: Presentazione offerte: priorità configurabile e ordering con propensity
Categoria: FUNZIONALE
Priorita: medium
Le offerte devono essere presentate secondo una priorità configurabile dall'utente marketing per prodotto singolo o bundle, oltre ad essere associabili a specifici touchpoint (REQ-F-005). Il sistema deve poter integrare modelli di propensity per ordinare automaticamente le offerte in base alla probabilità di interesse del cliente.

Acceptance Criteria

AC-1È possibile definire priorità configurabili per offerte/bundle e per touchpoint.
AC-2Il sistema supporta l'ordering automatico basato su modelli di propensity.
AC-3L'ordine risultante è tracciato nei log di proposta per eventuale audit.
